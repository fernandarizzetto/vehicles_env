# vehicles_env

Aplicação web desenvolvida com Streamlit para análise exploratória de dados de anúncios de veículos usados nos Estados Unidos.
link para aplicação web: https://vehicles-env-t7pp.onrender.com

## Funcionalidades

- Visualização da distribuição da preços dos veículos
- Gráfico de dispersão entre preço e quilometragem
- Interface interativa para exploração dos dados

## Tecnologias

- Python
- Pandas
- Streamlit
- Plotly Express

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
